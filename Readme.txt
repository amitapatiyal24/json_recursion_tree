To run this program open "tree-view" with Google Chrome.

Program description:
It reads the json input from "treeview.js" file and converts it into a nested json structure using recurive call. 
The fuction "createTreeDom" displays it in the form of a tree structure on the screeen using recursion.

If we need to read the input from the backend insted of a file it can be achieved by making an ajax call to the get API.
